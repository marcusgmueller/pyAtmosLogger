Installation
============

Requirements
------------
1. Install Python 3
2. Install Python Package Manager pip

via pip
-------

.. code-block:: console

   pip install "pyAtmosLogger @ git+https://github.com/marcusgmueller/pyAtmosLogger"



via Git
-------
1. clone repository

.. code-block:: console

   git clone https://github.com/marcusgmueller/pyAtmosLogger.git

2. install pyAtmosLogger

.. code-block:: console

   pip install .
